this contains angular code

to create an Angular App
ng new *name of the app*

To start the Angular App
ng serve 
to start the angular app in a specfic port
ng serve --port 8999

to generate components
ng g c *name of the component*
eg: ng g c Components(folder name in src/app)/home

to install json server: npm i json-server

to start json server: json-server --watch db.json --port 5000

to create a service: ng g s *name of the service*
eg: ng g s Services(folder name in src/app)/student

to create Pipe: ng g pipe *name of the pipe*
eg: ng g pipe Pipes(folder name in src/app)/ellipsis

to install ngrx store
npm install @ngrx/store
